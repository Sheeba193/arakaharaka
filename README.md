# Arakaharaka Enterprises Website 🌍

A modern, responsive portfolio website for Arakaharaka Enterprises, an import and export company based in Kenya.

## Live Demo

Live site: https://arakaharakaenterprise.com

## Features

- Responsive design
- Services, partners, testimonials
- Contact via email, WhatsApp, and social media
- Clean UI with animations

## Project Structure

- `index.html` — Homepage
- `thank-you.html` — Post-submission / confirmation page
- `assets/` — CSS, images, videos
- `js/` — JavaScript (including `main.js`, `emailjs.min.js`)

## Run Locally

Quick options to preview the site locally:

- Open `index.html` directly in your browser (double-click).
- Serve over a local HTTP server (recommended for JS features):

	- Python 3:

		```bash
		python -m http.server 8000
		```

	- Node (http-server):

		```bash
		npx http-server -p 8000
		```

Then visit `http://localhost:8000` in your browser.

## Deployment / Hosting Notes

- The live demo is hosted at the URL above — ensure DNS and SSL are configured for `arakaharakaenterprise.com`.
- If using a static host (Netlify, Vercel, GitHub Pages), point the domain to the host and publish the repository's static files.
